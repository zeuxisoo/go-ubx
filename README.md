# Go-Ubx

A console application for checking the target ubx status

## Develop

Create directory

    mkdir go-ubx && cd $_
    mkdir src/github.com/zeuxisoo && cd $_

Clone it first

    git clone https://github.com/zeuxisoo/go-ubx.git
    cd go-ubx

Run it

    go run *.go -e <EVENT_ID>

Build it

    make build
